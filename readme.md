
## Crunch REPL

**Getting Started**

  npm install crunch-repl

You need to require crunch-repl in your file First

  file.js
  const repl = require('crunch-repl').start();

Go CLI and Run:
  node file.js

You should have:
  crunch-$

To see Version of REPL You are running, Run:
  crunch-$ version
